# Student Management System

## 📌 Description
This is a Java-based console application developed to manage student records.
It allows users to add, view, and delete student details using basic OOP concepts.

## 🚀 Features
- Add Student
- View Students
- Delete Student

## 🛠 Technologies Used
- Java
- ArrayList
- OOP Concepts

## ▶️ How to Run
1. Compile the program:
   javac Main.java

2. Run the program:
   java Main
